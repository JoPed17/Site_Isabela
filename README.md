# Site Isabela Batista

Site estático para apresentação de coloração pessoal e consultoria de imagem.

## Estrutura
- `index.html`: página principal com seções Hero, Sobre, Quem sou eu, Serviços, Depoimentos e Contato.
- `style.css`: estilos complementares.
- `heroImage/` e assets: imagens utilizadas nas seções.

## Desenvolvimento
- Usa Tailwind via CDN.
- Ajustes de layout focados em mobile (redução de espaçamentos, harmonização de imagens, ocultar imagem em "Sobre" no mobile).

## Publicação
Pode ser hospedado no GitHub Pages ou em qualquer serviço estático.
